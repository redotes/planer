계획 플래너 사이트 기획안 

1. 필요한 웹페이지가 몇개가 있어야 하는가? 
- 사이트를 소개할 메인 페이지
- 로그인을 위한 로그인 페이지 
- 플래너를 적을 페이지 
- 마이페이지 
- 회원가입 페이지 
- 디자인 선택 카테고리 
디자인 참조할 사이트 : 미리캔버스(PPT 만드는 사이트) / 타 플래너 사이트를 통해 확인하기 
2. 만들고자 하는 기능이 몇개가 있는가?
- > 플래너를 적는 기능
- > 적은 데이터를 자동 저장하도록 설정 
- > 만든 데이터를 전부 엑셀에 들어가도록 설정 (디자인 포함)
- > 시간이 되면 알림이 울리도록 설정하고 싶다 
3. 만들려고 하는 사유가 무엇인가? 
- > 오늘의 할일을 적고 싶은데 적은 내용을 저장해서 내가 한 내용도 
함께 확인을 할 수 있는 방법이 없을까 생각했다
- > 할일이 많은 상황에 필수로 해야하는 시간대에 알림이 울리면 
	상황을 인지하고 중요한 일들을 할 수 있기 때문에 생각했다
4. 이용을 하고 싶은 언어는 몇개인가? 
	1. react+ plugin로 프론트를 구축해서 활용하기 
	2. 파이썬을 이용해서 디자인과 데이터 전부를 엑셀로 이동시키기 
	3. 알람 ( 시간 설정을 프론트로 알람이 울리게하는건 파이썬으로 백앤드를 활용)
5. 데이터 저장을 어디서 할 것이며 어떤 방법으로 다른 곳으로 이동 할 것인가?
	- 저장 위치는 정확히 어떻게 하는지 몰라서 지정을 못함 추후 백앤드 시도 시 알아보기 
	- 데이터 이동 및 저장은 백앤드 코드를 짤때 해야한다고 한다 
	따라서 파이썬을 이용해서 완성을 할때 사용할 예정이다
	
6. 디자인은 어떤 방법으로 할껀가?
	디자인 만드는 내용의 경우 커스터 마이징으로 진행하며 
	초기 엑셀을 이용해서 만든 내용을 넣어두면 될 것 같음
	테스트 초기이므로 4개에서 5개 대충 만들어보기 

7. chat gpt를 이용을 많이 할껀가?
	1. 이용할 언어를 알기 위해서 사용했으며 추가적으로 백앤드 코드를 짤때 
	에러 확인 목적으로 이용하면 좋을 것 같음
	2. 코드를 직접 묻는 건 최대한 내가 진행 해보고 안됬을 때 이야기 해보자
8. 깃허브 등록을 반드시 할껀지?
	반드시 진행을 해야한다 왜냐하면 집에서도 해야하니까
